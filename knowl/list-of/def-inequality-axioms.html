Definition 4.6. Axioms of Inequality.

Let \(a\text{,}\) \(b\text{,}\) and \(c\) be real numbers. The axioms of inequality state that
  1. If \(a \gt b\text{,}\) then \(b \lt a\text{.}\)
  2. If \(a \gt b\text{,}\) then \(a + c \gt b + c\text{.}\)
  3. If \(a \lt b\text{,}\) then \(a - c \lt b - c\text{.}\)
  4. If \(a \gt b\) and \(c \gt 0\text{,}\) then \(ac \gt bc\text{.}\)
  5. If \(a \gt b\) and \(c \gt 0\text{,}\) then \(\frac{a}{c} \gt \frac{b}{c}\text{.}\)
